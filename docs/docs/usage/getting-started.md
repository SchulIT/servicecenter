# Erste Schritte

Nach der Installation müssen Räume, Gerätetypen, Geräte und Problemtypen angelegt werden. Dies geschieht über das 
Verwaltungsmenü, indem die entsprechenden Punkte angeklickt werden.

## Beispiele

Zur Inspiration einige Beispiel aus dem Schulalltag.

### Gerätetypen

* Beamer
* Computer
* Drucker
* Dokumentenkamera
* Laptop
* Leinwand
* Soundbar
* Tablet

### Problemtypen

* Beamer
  * Ausgabe
  * Fernbedienung
  * Screen Mirroring
  * Sonstiges
* Computer
  * Anmeldung
  * Anwendungen
  * Medienwiedergabe
  * Peripherie
  * Sonstiges
* Laptop
  * Anmeldung
  * Anwendungen
  * WLAN
* Drucker
  * Ausdruck
  * Tinte/Toner
  * Sonstiges
