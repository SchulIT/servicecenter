Übersicht über IT im päd. Netzwerk mit Möglichkeiten zur Statusübersicht und Problemverwaltung. 

## Überblick

Mit dem ServiceCenter kann der aktuelle Status des pädagogischen Netzwerkes festgehalten werden. Lehrkräfte oder Mitarbeitende
können über das Tool Probleme mit der schulischen IT-Infrastruktur melden, welche bspw. von der mit der IT betrauten Lehrkraft
gelöst werden können.
